Library DB 1.0:
-----------------------------

1) Registration

- This is intended for first time users to create a library account and use it to check out/return books. Upon registering, the user will choose
a login name, and they will be given an ID number to use when logging in in the future.

2) Login

- Allows the user to login using the login name/ID number provided after registering.

3) Search and Check Out Books

- Allows the user to search for books either by Title, Author or Category. After searching, the user can choose to check out a book for 1 month

4) See Currently Checked Out Books

- Presents the user with a list of their currently checked out books, and their respective due dates.

5) Return a Book

- Allows the user to return a currently checked out book.
